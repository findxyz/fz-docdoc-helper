# docdochelper
## 简介
#### docdochelper是一款辅助[docdoc](https://github.com/github20120522/docdoc)使用的客户端gui软件
## 目的
+ #### 自动区分前端发起的请求是该请求正常接口还是请求模拟数据
## 说明
+ #### 支持npm run开发方式下的跨域请求，支持的跨域地址为http://localhost:8080
